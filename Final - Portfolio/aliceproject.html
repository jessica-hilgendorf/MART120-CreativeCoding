A Walk in the Woods

For this project, I used the program Alice to create an interactive story about a bear cub exploring the woods. The user can control the bear cub with the arrow keys and explore the wilderness. There are a couple of animals the bear cub can encounter and talk to. I really enjoyed being able to use the multiple "classes" available in Alice to create the wilderness scene.
