Play Lab Witchy Game

Using Play Lab on Code.org, I created a game. The user plays as the witch. Using the arrow keys, the witch moves up and down. The left arrow key shoots a fire ball at the dog and cat moving around the screen. With each successful collision with the dog, a point is earned. However, every time a fireball hits the cat, a point is taken away.
